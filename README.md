# 网络故障排查工具

一个功能全面的网络故障排查工具，基于Python和PyQt5开发，专为网络工程师设计。

## 功能特性

### 1. 网络连通性测试
- **Ping测试**：测试目标主机的连通性和响应时间
- **路由追踪**：追踪到目标主机的网络路径

### 2. 端口扫描和服务检测
- **端口扫描**：扫描指定范围的端口
- **服务识别**：自动识别常见服务和Banner抓取

### 3. 网络速度测试
- **带宽测试**：测试网络下载和上传速度
- **延迟测试**：测试网络延迟

### 4. 网络设备配置管理
- **IP配置查看**：查看网络适配器的IP配置
- **DNS缓存管理**：刷新DNS缓存

### 5. 流量分析和监控
- **实时流量监控**：监控网络上传下载速度
- **网络连接查看**：查看当前活动的网络连接

### 6. DNS查询和分析
- **DNS记录查询**：支持A、AAAA、MX、NS、TXT等多种记录类型
- **域名解析**：解析域名的IP地址

### 7. Wi-Fi信号强度检测
- **Wi-Fi扫描**：扫描周围的Wi-Fi网络
- **信号强度检测**：显示Wi-Fi信号强度

### 8. SNMP管理
- **SNMP查询**：查询设备的SNMP信息
- **设备信息获取**：获取网络设备的基本信息

### 9. 网络拓扑
- **拓扑生成**：基于路由追踪生成网络拓扑图
- **可视化展示**：以流程图形式展示网络结构

### 10. 网络诊断
- **批量Ping**：批量测试多个IP地址的连通性
- **综合诊断**：全面诊断网络连接状态

### 11. 智能助手
- **AI问答**：基于豆包大模型的智能网络故障诊断
- **实时建议**：提供网络问题的解决方案

### 12. 抓包分析
- **实时抓包**：捕获网络数据包
- **协议过滤**：支持TCP/UDP/ICMP等协议过滤
- **端口过滤**：按端口过滤数据包
- **统计分析**：实时显示数据包数量和字节数

### 13. HTTP监控
- **HTTP代理**：本地HTTP代理服务器
- **请求监控**：监控HTTP/HTTPS请求和响应
- **性能分析**：显示请求耗时和响应大小
- **状态码分析**：根据状态码标记请求状态

## 安装说明

### 系统要求
- Python 3.6+
- Windows/Linux/macOS

### 方式一：直接运行（推荐开发者）

1. 克隆或下载项目
```bash
git clone <repository_url>
cd network-diagnostic-tool
```

2. 安装依赖
```bash
pip install -r requirements.txt
```

3. 运行程序
```bash
python main.py
```

### 方式二：Windows打包版本（推荐普通用户）

#### 方式A：绿色免安装版本（文件夹）

1. 下载项目源码
2. 在Windows系统中双击运行 `build_windows.bat`
3. 等待打包完成（约2-5分钟）
4. 打开 `dist\网络故障排查工具` 文件夹
5. 双击 `网络故障排查工具.exe` 运行
6. 可以将整个文件夹复制到其他电脑使用

**优点：**
- 启动速度快
- 文件体积小
- 绿色免安装

#### 方式B：单文件版本

1. 下载项目源码
2. 在Windows系统中双击运行 `build_onefile.bat`
3. 等待打包完成（约5-10分钟）
4. 打开 `dist` 文件夹
5. 双击 `网络故障排查工具.exe` 运行
6. 可以将exe文件复制到任何地方使用

**优点：**
- 只有一个exe文件
- 方便分发
- 无需其他文件

**缺点：**
- 文件较大（约50-100MB）
- 首次启动较慢

### 注意事项
- Windows用户：首次运行可能需要防火墙允许
- macOS用户：可能需要在系统偏好设置中允许运行
- Linux用户：确保有执行权限 `chmod +x main.py`

## 使用说明

### Ping测试
1. 选择"Ping测试"标签页
2. 输入目标主机（IP地址或域名）
3. 设置Ping次数
4. 点击"开始Ping"

### 端口扫描
1. 选择"端口扫描"标签页
2. 输入目标主机
3. 设置扫描的端口范围
4. 点击"开始扫描"

### 速度测试
1. 选择"速度测试"标签页
2. 点击"开始速度测试"
3. 等待测试完成

### 流量监控
1. 选择"流量监控"标签页
2. 点击"开始监控"开始实时监控
3. 点击"停止监控"停止监控
4. 点击"查看网络连接"查看当前连接

### DNS查询
1. 选择"DNS查询"标签页
2. 输入域名或IP地址
3. 选择记录类型
4. 点击"查询"或"解析域名"

### Wi-Fi扫描
1. 选择"Wi-Fi扫描"标签页
2. 点击"扫描Wi-Fi网络"

### SNMP管理
1. 选择"SNMP管理"模块
2. 输入设备IP地址
3. 输入Community字符串（默认为public）
4. 输入OID或点击"获取设备信息"

### 网络拓扑
1. 选择"网络拓扑"模块
2. 输入目标IP地址
3. 设置最大跳数（1-5）
4. 点击"生成拓扑图"

### 网络诊断
1. 选择"网络诊断"模块
2. 点击"开始诊断"进行综合诊断
3. 或在批量Ping区域输入多个IP地址进行批量测试

### 智能助手
1. 选择"智能助手"模块
2. 配置豆包API Key（可选，首次使用需要配置）
3. 输入网络问题或选择建议问题
4. 点击"发送"获取AI回答

### 抓包分析
1. 选择"抓包分析"模块
2. 选择协议过滤（可选）
3. 输入端口过滤（可选）
4. 点击"开始抓包"
5. 查看实时捕获的数据包信息

### HTTP监控
1. 选择"HTTP监控"模块
2. 设置代理端口（默认8080）
3. 输入主机过滤（可选）
4. 点击"启动代理"
5. 在浏览器中设置HTTP代理为显示的地址
6. 浏览网页，查看监控到的HTTP请求
