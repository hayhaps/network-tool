# 网络故障排查工具

一个功能全面的网络故障排查工具，基于Python和PyQt5开发，专为网络工程师设计。

## 功能特性

### 1. 网络连通性测试
- **Ping测试**：测试目标主机的连通性和响应时间
- **路由追踪**：追踪到目标主机的网络路径

### 2. 端口扫描和服务检测
- **端口扫描**：扫描指定范围的端口
- **服务识别**：自动识别常见服务和Banner抓取

### 3. 网络速度测试
- **带宽测试**：测试网络下载和上传速度
- **延迟测试**：测试网络延迟

### 4. 网络设备配置管理
- **IP配置查看**：查看网络适配器的IP配置
- **DNS缓存管理**：刷新DNS缓存

### 5. 流量分析和监控
- **实时流量监控**：监控网络上传下载速度
- **网络连接查看**：查看当前活动的网络连接

### 6. DNS查询和分析
- **DNS记录查询**：支持A、AAAA、MX、NS、TXT等多种记录类型
- **域名解析**：解析域名的IP地址

### 7. Wi-Fi信号强度检测
- **Wi-Fi扫描**：扫描周围的Wi-Fi网络
- **信号强度检测**：显示Wi-Fi信号强度

### 8. SNMP管理
- **SNMP查询**：查询设备的SNMP信息
- **设备信息获取**：获取网络设备的基本信息

### 9. VLAN配置
- **VLAN信息查看**：查看系统中的VLAN配置

## 安装说明

### 系统要求
- Python 3.6+
- Windows/Linux/macOS

### 安装步骤

1. 克隆或下载项目
```bash
git clone <repository_url>
cd network-diagnostic-tool
```

2. 安装依赖
```bash
pip install -r requirements.txt
```

3. 运行程序
```bash
python main.py
```

## 使用说明

### Ping测试
1. 选择"Ping测试"标签页
2. 输入目标主机（IP地址或域名）
3. 设置Ping次数
4. 点击"开始Ping"

### 端口扫描
1. 选择"端口扫描"标签页
2. 输入目标主机
3. 设置扫描的端口范围
4. 点击"开始扫描"

### 速度测试
1. 选择"速度测试"标签页
2. 点击"开始速度测试"
3. 等待测试完成

### 流量监控
1. 选择"流量监控"标签页
2. 点击"开始监控"开始实时监控
3. 点击"停止监控"停止监控
4. 点击"查看网络连接"查看当前连接

### DNS查询
1. 选择"DNS查询"标签页
2. 输入域名或IP地址
3. 选择记录类型
4. 点击"查询"或"解析域名"

### Wi-Fi扫描
1. 选择"Wi-Fi扫描"标签页
2. 点击"扫描Wi-Fi网络"

### SNMP管理
1. 选择"SNMP管理"标签页
2. 输入设备IP地址
3. 输入Community字符串（默认为public）
4. 输入OID或点击"获取设备信息"

3. **网络环境**：部分功能需要网络连接
4. **系统兼容性**：不同操作系统的某些功能可能有所差异
